# Веб-приложение для обработки и поиска изображений

## Описание проекта
Это веб-приложение представляет собой современную систему для загрузки, обработки и поиска изображений. Приложение автоматически обнаруживает текст и объекты на загруженных изображениях с использованием передовых технологий компьютерного зрения.

## Основные функции
1. Загрузка изображений
   - Поддержка различных форматов изображений
   - Автоматическое определение текста с помощью EasyOCR
   - Распознавание объектов с использованием YOLOv8

2. Поиск и навигация
   - Поиск по обнаруженному тексту на изображениях
   - Поиск по распознанным объектам
   - Автодополнение при вводе поисковых запросов
   - Удобный просмотр изображений в виде сетки

3. Управление данными
   - Хранение метаданных изображений в базе данных SQLite
   - Возможность скачивания обработанных изображений
   - Эффективная система поиска и фильтрации

## Техническая реализация
### Бэкенд (Backend)
- Фреймворк: Flask (Python)
- Системы распознавания: YOLOv8 и EasyOCR
- База данных: SQLite
- RESTful API для взаимодействия с фронтендом

### Фронтенд (Frontend)
- Фреймворк: Vue.js
- Сборщик: Vite
- Современный и отзывчивый пользовательский интерфейс

## Структура проекта
- backend/ - Серверная часть на Flask
- frontend/ - Клиентская часть на Vue.js
- src/ - Основной код приложения

## API Endpoints
- POST /api/upload - Загрузка и обработка изображений
- GET /api/images - Получение списка изображений с параметрами поиска
- GET /api/images/<id> - Скачивание конкретного изображения
- GET /api/suggestions - Получение поисковых подсказок

## Требования к установке
1. Python с установленными зависимостями из requirements.txt
2. Node.js для работы с фронтендом
3. Виртуальное окружение Python для изоляции зависимостей

Это приложение предоставляет удобный и современный интерфейс для работы с изображениями, комбинируя мощные возможности компьютерного зрения с удобством веб-интерфейса.
